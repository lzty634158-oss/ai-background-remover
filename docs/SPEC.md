# AI 背景去除器 - MVP 需求文档

## 1. 产品概述

| 项目 | 内容 |
|------|------|
| **产品名称** | AI 背景去除器 (AI Background Remover) |
| **产品定位** |面向全球用户的在线图片背景去除工具 |
| **核心功能** | 上传图片，自动去除背景，支持多语言 |
| **商业模式** | 免费增值 (Freemium) - 免费10张，之后每张 $0.5 |
| **目标用户** | 设计师、电商卖家、内容创作者、普通用户 |

---

## 2. 功能需求

### 2.1 核心功能 (MVP)

| 优先级 | 功能 | 描述 |
|--------|------|------|
| P0 | 图片上传 | 支持拖拽和点击上传 PNG/JPG/WebP，单文件最大 10MB |
| P0 | 背景去除 | 调用 Remove.bg API 去除图片背景 |
| P0 | 结果下载 | 支持下载去除背景后的 PNG 图片 |
| P0 | 多语言 | 支持 7 种语言自动切换 + 手动切换 |
| P1 | 免费额度 | 新用户免费 10 张，超出引导付费 |
| P1 | 用户注册 | 邮箱注册登录，保存使用记录 |
| P1 | 余额充值 | 用户可购买积分包 |

### 2.2 用户流程

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  访问首页     │ ──▶ │  上传图片     │ ──▶ │  处理图片    │
└──────────────┘     └──────────────┘     └──────────────┘
                                                │
                                                ▼
                     ┌──────────────┐     ┌──────────────┐
                     │  下载结果     │ ◀── │  展示结果    │
                     └──────────────┘     └──────────────┘
```

### 2.3 权限控制

| 用户状态 | 免费额度 | 超出后行为 |
|----------|----------|------------|
| 未登录 | 0 张 | 引导注册 |
| 已登录未付费 | 10 张 | 引导充值 |
| 已付费 | 余额用完 | 引导充值 |

## 3. 技术架构

### 3.1 系统架构

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   前端      │ ──▶ │   后端 API  │ ──▶ │ Remove.bg  │
│  (HTML/JS)  │     │  (FastAPI)  │     │   API      │
└─────────────┘     └─────────────┘     └─────────────┘
                           │
                           ▼
                    ┌─────────────┐
                    │  数据库     │
                    │ (SQLite)    │
                    └─────────────┘
```

### 3.2 技术栈

| 层级 | 技术选型 |
|------|----------|
| 前端 | HTML5 + 原生 JavaScript |
| 后端 | Python FastAPI |
| 数据库 | SQLite (轻量级) |
| 图片处理 | Remove.bg API |
| 支付 | Stripe (国际) / 支付宝 (国内) |

### 3.3 目录结构

```
bg-remover/
├── app/
│   ├── main.py           # FastAPI 主程序
│   ├── models.py         # 数据库模型
│   ├── auth.py           # 用户认证
│   ├── payment.py       # 支付处理
│   └── static/
│       ├── index.html    # 前端页面
│       ├── style.css     # 样式
│       └── app.js        # 前端逻辑
├── data/                 # 数据库文件
└── requirements.txt     # 依赖
```
